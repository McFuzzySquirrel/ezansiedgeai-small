# Sub-Agent Contributions

## Sub-Agent: project-architect
- **Task delegated:** P0-101 — Complete Android app scaffold with DI, navigation, and core interfaces
- **Decisions made:** Manual DI with lazy {} singletons instead of Hilt/Dagger (zero reflection, zero code-gen); Material Icons Extended for bottom bar icons (R8 strips unused)
- **Alternatives considered:** Hilt (rejected — reflection overhead, GMS concerns), Koin (rejected — runtime overhead), manual object graph (selected)
- **Outcome:** 20 Kotlin files total (up from 3), fully wired navigation, all core interfaces defined
- **Handoff to other agents:** Core interfaces consumed by learner-data-engineer, content-pack-engineer, ai-pipeline-engineer

## Sub-Agent: learner-data-engineer
- **Task delegated:** P0-105 — Learner profile system with encryption, multi-profile, chat history
- **Decisions made:** AES-256-GCM with Android Keystore; atomic file writes (temp→fsync→rename); org.json.JSONObject over kotlinx.serialization (zero extra deps); WAL mode for chat SQLite; field-level encryption for chat messages
- **Alternatives considered:** Room (rejected — overkill for simple schemas), SharedPreferences (rejected — not encrypted), EncryptedSharedPreferences (rejected — limited to key-value)
- **Outcome:** 9 new files, full profile CRUD with cascading delete, encrypted chat history, default preferences
- **Handoff to other agents:** ProfileRepository and PreferenceRepository consumed by ai-pipeline-engineer (prompt preferences) and android-ui-engineer (profile UI)

## Sub-Agent: content-pack-engineer
- **Task delegated:** P0-103 — Content pack loader with SHA-256 verification and SQLite access
- **Decisions made:** Double verification (before and after copy); pure-Kotlin cosine similarity as fallback; read-only SQLite flags; topic tree built from flat paths
- **Alternatives considered:** Room (rejected), FAISS JNI (deferred to ai-pipeline-engineer), single verification (rejected — catches storage corruption)
- **Outcome:** 5 new files, full pack lifecycle (install/uninstall/query), embedding access for AI pipeline
- **Handoff to other agents:** PackDatabase.getAllEmbeddings() and getChunkById() consumed by ai-pipeline-engineer for retrieval
